# Node-JS-Socket-A4-E6

## Prérequis

- node 18.0
- npm ^8.5.0

## Démarrer le projet

A la racine du projet :

- npm i
- npm start

##DB

We use a postgreSQL database by heroku service.

To launch the project and enable database connection you should use this command line :
```
DATABASE_URL=***** node index.js
```
